# Database Browser

OpenDeploy provides browser-based database management.

## PostgreSQL

- Database list
- Schema list
- Table list
- Views, functions, sequences and extensions
- Columns, indexes, constraints and foreign keys
- Table data grid
- Insert, edit and delete rows
- SQL editor
- EXPLAIN and EXPLAIN ANALYZE
- CSV/JSON export
- Query history

## MySQL / MariaDB

- Database list
- Table list
- Columns, indexes and foreign keys
- Stored procedures and triggers
- SQL editor
- EXPLAIN query
- Process list
- Kill query
- Slow query log viewer

## Redis

- Key list and search
- Type viewer
- String, hash, list, set and sorted set editor
- TTL view/edit
- Memory usage
- Redis info and slowlog

## SQLite

- SQLite file selector
- Table browser
- Query editor
- Import/export

## Query Safety

- Query timeout
- Maximum result limit
- Dangerous operation confirmation
- `DELETE` without `WHERE` warning
- Query audit logging
- Role based execution permissions
